# Mal D'Aria 2021
Prova del sito Mal D'Aria

Tool utilizzati:
* Webpack
* Sass
* Bootstrap
* Tailwind

# Installazione
Prima installa le dipendenze. Vai sul terminale:
```shell
npm install
```

Per avviare il progetto in Development Mode:
```shell
npm start
```

Per buildare il sito:
```shell
npm run build
```
